YÜZ KİLİT SİSTEMİ

Yüz kilit sistemi için geliştirdiğim yazılım, yüz tanıma teknolojisi kullanarak güvenli ve hızlı bir erişim sağlıyor. Kullanıcı dostu arayüzü ve yüksek doğruluk oranı ile sistem, güvenlik standartlarını en üst düzeye çıkarıyor. Yazılım, çeşitli ortamlarda etkili bir biyometrik güvenlik çözümü sunarak kullanıcıların günlük yaşamlarını kolaylaştırıyor. Yüz kilit sistemi yazılımım, modern güvenlik ihtiyaçlarına uygun olarak tasarlanmıştır.

KOD ÖRNEĞİ

import cv2
face_cascade = cv2.CascadeClassifier
(cv2.data.haarcascades + 
"haarcascade_frontalface_default.xml")
cap = cv2.VideoCapture(0)
while True:
ret, frame = cap.read()
if not ret:
break
frame = cv2.flip(frame, 1)
gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)
faces = face_cascade.detectMultiScale
(gray, scaleFactor=1.1, minNeighbors=5, 
minSize=(30, 30))
for (x, y, w, h) in faces:
center_x = x + int(w / 2) 
center_y = y + int(h / 2) - int(h / 4)  
cv2.circle(frame, (center_x, center_y), 
5, (0, 0, 255), -1)
cv2.putText(frame, "Hedefe Kilitlendi", 
(x, y - 20), cv2.FONT_HERSHEY_SIMPLEX, 
0.7, (0, 0, 255), 2)
cv2.imshow("Yüz Tespiti", frame)
if cv2.waitKey(1) & 0xFF == ord('q'):
break
cap.release()
cv2.destroyAllWindows()
